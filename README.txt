Compile computeDFAs and run the following commands...

To test complementation:
- example:
    java computeDFAs -c D1.txt

To test intersection:
- example:
    java computeDFAs -i D1.txt D2.txt

To test symmetric difference:
- example:
    java computeDFAs -s D1.txt D2.txt

To test non-emptiness:
- example:
    java computeDFAs -ne D1.txt

To test equivalence:
- example:
    java computeDFAs -e D1.txt D2.txt

NOTE: D1.txt/D2.txt can be replaced with any other test files.
